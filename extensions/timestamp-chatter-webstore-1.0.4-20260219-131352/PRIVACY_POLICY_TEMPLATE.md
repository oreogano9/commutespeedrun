## Timestamp Chatter Privacy Policy

Last updated: [DATE]

Timestamp Chatter helps users surface timestamped YouTube comments as on-video overlays and timeline markers.

### Data We Access
- YouTube page context (video ID, playback time, player UI state)
- Public YouTube comment data required for functionality (comment text, author display name, avatar URL, likes, timestamps)
- Local extension settings (overlay preferences, filters, toggles)

### How Data Is Used
- To fetch and process timestamped comments for the current YouTube video
- To render notification overlays and timeline markers
- To persist your customization settings between sessions

### Data Sharing
- We do not sell personal data.
- We do not share data with third-party analytics or ad networks.
- Comment and video data is requested from YouTube endpoints only to provide extension functionality.

### Data Storage
- Settings are stored using Chrome extension storage.
- No separate developer-operated backend is used for user profile tracking.

### User Controls
- You can disable overlays using the in-player eye toggle.
- You can change filtering and display settings in the extension popup.
- You can remove the extension at any time from Chrome/Arc extension settings.
